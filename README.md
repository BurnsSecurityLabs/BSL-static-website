# Static Website Project (GitHub Pages)

This repository contains a **simple static website** used for practice and iteration with HTML, CSS, JavaScript, and GitHub Pages.

## Overview

The project represents a personal-style static site with multiple pages and basic styling. It is primarily used to:

* Practice front-end fundamentals
* Refresh GitHub workflows
* Test GitHub Pages deployment via Actions

## Project Structure

```text
.github/workflows/
  └─ static.yml        # GitHub Actions workflow for deployment

index.html             # Landing / home page
about.html             # About page
work.html              # Work / projects page
resume.html            # Resume page
style.css              # Global styles
script.js              # Basic JavaScript functionality
```

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* GitHub Actions
* GitHub Pages

## Deployment

This site is deployed automatically using **GitHub Actions**.

* Any update pushed to the configured branch triggers the workflow
* The workflow builds and publishes the static site to GitHub Pages

## Purpose of This Repository

* Practice building and updating a static website
* Experiment with layout, styling, and structure
* Learn and reinforce GitHub Pages + Actions usage
* Serve as a sandbox, not a final product

## Notes

* Content and structure may change frequently
* Commits prioritize learning over polish
* No guarantee of stability or completeness
